Why is the viewport meta tag important?

The viewport meta tag is important because it controls how a webpage is displayed on mobile devices and affects how it scales. In responsive web design, it helps ensure that a webpage is optimized for different screen sizes. Without it, a site might appear zoomed out or improperly scaled on mobile devices.

How can you adjust the CSS for printing?

You can adjust the CSS for printing using the media print rule, which allows you to apply styles specifically for when a user prints a webpage, ensuring that the layout is appropriate for paper.

When would you want to adjust the CSS for printing?

You would want to adjust the CSS for printing when you need to optimize the content for a printed format, hiding unnecessary elements like navigation bars and ensuring readability and proper layout on paper.
