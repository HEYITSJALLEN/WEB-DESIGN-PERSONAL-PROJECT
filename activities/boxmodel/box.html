1. What is the difference between block tags and inline tags.

  • Block Tags: Block elements take up the full width available and start on a new line. They typically contain other block or inline elements.
  • Inline Tags: Inline elements only take up as much width as their content and do not start on a new line. They can be nested inside block elements and typically do not contain block elements.

2. What is the margin, border, and padding?

  • Margin: The space outside an element's border. It creates distance between the element and other elements around it.
  • Border: The boundary that surrounds an element. It can be styled with various widths, colors, and styles (solid, dashed, etc.).
  • Padding: The space between an element's content and its border. It creates distance within the element itself.
