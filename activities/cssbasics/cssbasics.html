Explain the difference between inline, embedded, and external CSS.

Incline - Defined directly within an HTML element using the style attribute.

Embedded - Placed within the tag in the head section of an HTML document.

External CSS - Defined in a separate .css file, which is linked to the HTML document using the link tag.

Explain why you should only use external CSS.

Because it promotes a clear separation between HTML structure and CSS presentation, making the code more organized, modular, and easier to maintain. An external CSS file can be reused across multiple pages, reducing redundancy and improving consistency. It also enhances maintainability, as global changes can be applied easily from a single location. Additionally, external CSS improves performance since browsers cache the files, leading to faster load times for users navigating multiple pages.

What is a selector used for?

In HTML, a selector is used to change the style of a specific type of element by selecting it and applying a set of CSS rules. Selectors are the names given to styles in internal and external style sheets.
