1. List out the different ways to select an item.

By using ID, Class, Tag, Attribute, Pseudo-classes, Query Selector.

2. Difference Between an ID and a Class?

    ID

    1. Unique identifier for a single element on a page.
    2. Used for identifying one specific element.
    3. Can only be applied to one element at a time.

    Class

    1. Can be applied to multiple elements.
    2. Useful for styling groups of elements that share similar characteristics.

3. What is the difference between a SPAN tag and a DIV tag?

    SPAN TAG

    1. Inline element.
    2. Does not cause line breaks and only takes up the space of its content.
    3. Used to style specific parts of a line of text (e.g., applying a style to a word).

    DIV TAG

    1. Block-level element.
    2. Causes line breaks before and after the element.
    3. Typically used to group larger chunks of content or structure sections of a page.

4. What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

The div tag is a generic block-level container without specific meaning, often used for layout or structural purposes. In contrast, header, main, and footer are semantic elements that represent introductory content, the main content, and closing elements, respectively. These elements help improve accessibility and SEO by providing a meaningful structure to the content, while div is primarily a layout tool.

5. What are different ways to select a color?

There are several ways to select colors in CSS: hexadecimal (hex) codes, RGB values, RGBA values with transparency, HSL (Hue, Saturation, Lightness) values, HSLA values with transparency, and named colors.
